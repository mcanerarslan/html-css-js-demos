# html-css-js-demos

Small collection of standalone HTML/CSS demos that show off landing experiences, forms, and layout practice exercises.

## Demos
- [MasterCard credit card layout](https://mcanerarslan.github.io/html-css-js-demos/credit-card.html) — interactive front/back credit card mockup plus a styled payment form.
- [Login panel](https://mcanerarslan.github.io/html-css-js-demos/login.html) — two-column welcome layout with hero image, credential inputs, and social sign-in buttons.
- [Portfolio page](https://mcanerarslan.github.io/html-css-js-demos/portfolio.html) — single-page profile with sections for introduction, skills, projects, and contact links.
